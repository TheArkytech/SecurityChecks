# SecurityChecks

Modular cybersecurity system for Arkytech's development team working with
AI-assisted coding (vibe coding). Designed to be executed by Claude.

## Quick Start

1. Copy `.npmrc` to each Node.js project
2. Install pre-commit hooks: `pre-commit install` (requires [gitleaks](https://github.com/gitleaks/gitleaks))
3. The Knowledge Base is ready at `docs/knowledge-base.md` — see [KNOWLEDGE-BASE-SETUP.md](docs/KNOWLEDGE-BASE-SETUP.md) for details

## Structure

```
SecurityChecks/
├── README.md
├── AGENTS.md                          # AI agent security rules
├── CLAUDE.md                          # Claude-specific repo instructions
├── .gitignore                         # Prevents accidental secret commits
├── .npmrc                             # npm hardening
├── .pre-commit-config.yaml            # gitleaks pre-commit hook
└── docs/
    ├── README.md                      # Architecture + cadence
    ├── PROMPT-A-RADAR.md              # Daily radar (Sonnet 4.6)
    ├── PROMPT-B-AUDITOR.md            # Weekly auditor (Opus 4.6)
    ├── PROMPT-C-VETTING.md            # On-demand vetting (Sonnet 4.6)
    ├── knowledge-base.md              # Live KB: threats, IOCs, decisions
    ├── KNOWLEDGE-BASE-SETUP.md        # KB guide
    ├── IMPROVEMENTS.md                # Security roadmap
    ├── SELF-IMPROVEMENT.md            # System auto-improvement log
    └── SCHEDULED-TASKS-SETUP.md       # Scheduled agents config
```

## Components

| Component | File | Model | Frequency | Description |
|---|---|---|---|---|
| Radar | [PROMPT-A-RADAR.md](docs/PROMPT-A-RADAR.md) | Sonnet 4.6 | Daily (automated) | Global cybersecurity scan |
| Auditor | [PROMPT-B-AUDITOR.md](docs/PROMPT-B-AUDITOR.md) | Opus 4.6 | Weekly / triggered | Source code security audit |
| Vetting | [PROMPT-C-VETTING.md](docs/PROMPT-C-VETTING.md) | Sonnet 4.6 | On-demand | Pre-install dependency evaluation |

## Hardening Included

- **`AGENTS.md`** — Supply chain security rules for AI agents (12 rules)
- **`CLAUDE.md`** — Repo-specific instructions for Claude
- **`.gitignore`** — Blocks `.env`, `*.key`, `*.pem`, credentials from commits
- **`.npmrc`** — `ignore-scripts=true`, `audit=true`, `audit-level=high`
- **`.pre-commit-config.yaml`** — Gitleaks v8.22.1 to detect secrets before commit
